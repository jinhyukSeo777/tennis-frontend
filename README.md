# GoGo Tennis

### 가상의 테니스장 예약 플랫폼

<a href="https://tennis-frontend-plum.vercel.app/" style="text-decoration: none;">GoGotennis 웹사이트</a>

<br>

## 🙌 프로젝트 소개

가상의 테니스장에 예약을 하고 익명의 사용자와 테니스 경기를 즐길 수 있습니다.

랭킹 및 커뮤니티 서비스를 통해 사용자 간 소통과 경쟁을 더욱 활발히 할 수 있습니다.
<br>
